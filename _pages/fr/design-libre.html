Design libres

Un logiciel libre est un logiciel dont le développement de son code source suit un processus ouvert. La licence du logiciel permet d’accéder librement à son code source, de créer des travaux dérivés et de le redistribuer.

Consultez la liste des logiciels libres utilisés par les administrations publiques canadiennes.

Vous pouvez ajouter un logiciel ou modifier ses informations en utilisant le Formulaire de Recherche Libre.

Besoin d’aide? Consultez la documentation sur comment utiliser la plateforme Échange de Ressources Ouvert.
